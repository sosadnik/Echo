# 04 — Instrukcje / prompty

Instrukcje operacyjne i prompty dla **agentów AI** pracujących nad projektem (implementacja,
audyt, utrzymanie dokumentacji).

Co tu trafia:
- system prompty i wytyczne dla agentów/subagentów,
- opisy powtarzalnych procedur uruchamianych przez agenta.

**To instrukcje *dla* agentów**, nie opis systemu. Opis systemu należy do `00`–`03`.

> Sam **proces pracy** (Analiza → Plan → Wdrożenie) realizują skille `analyze` / `plan-create` /
> `implement` oraz subagenci skonfigurowani natywnie dla używanej platformy. Reguły procesu:
> `CLAUDE.md` (Claude Code) lub `AGENTS.md` (Codex).
